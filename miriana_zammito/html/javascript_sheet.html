Comandi Javascript


Gli elementi più utilizzati in JavaScript per manipolare il DOM includono:

getElementById:Seleziona un elemento tramite il suo attributo id.
getElementsByClassName: Seleziona elementi tramite i loro nomi di classe.
getElementsByTagName: Seleziona elementi tramite il loro nome di tag (ad esempio, <p> o <div>).
querySelector: Seleziona il primo elemento che corrisponde a un selettore CSS specificato.
querySelectorAll: Seleziona tutti gli elementi che corrispondono a un selettore CSS specificato.
innerHTML: Modifica il contenuto HTML di un elemento.
innerText: Modifica il testo di un elemento.
classList: Gestisce le classi CSS di un elemento, permettendo di aggiungere, rimuovere, sostituire o verificare la presenza di una classe.
setAttribute: Imposta o modifica un attributo di un elemento.
removeAttribute: Rimuove un attributo da un elemento.
addEventListener: Aggiunge un gestore eventi a un elemento.
removeEventListener: Rimuove un gestore eventi da un elemento.


VARIABILI

In JavaScript, le variabili sono contenitori per memorizzare dati che possono essere di diversi tipi.

Tipi di Dati Primitivi:

Number: Rappresenta sia numeri interi che decimali. Esempio: let numero = 42;
String: Rappresenta una sequenza di caratteri. Esempio: let saluto = "Ciao";
Boolean: Rappresenta un valore logico vero o falso. Esempio: let flag = true;
Undefined: Indica che una variabile è stata dichiarata ma non è stata assegnata a nessun valore.
Null: Rappresenta un valore nullo o “nessun valore”.
Symbol: Utilizzato per creare identificatori univoci per le proprietà degli oggetti.
BigInt: Rappresenta numeri interi molto grandi che non possono essere rappresentati dal tipo Number.


Tipi di Dati di Riferimento:

Object: Rappresenta una collezione di proprietà, e una proprietà è un’associazione tra un nome (o chiave) e un valore.
Array: Un tipo speciale di oggetto usato per memorizzare collezioni ordinate di dati. Esempio: let lista = [1, 'due', true];
Function: Anche le funzioni in JavaScript sono oggetti, e hanno la capacità di essere invocate.


Dichiarazione di Variabili:

var: Dichiarazione di variabili con scope globale o di funzione.
let: Dichiarazione di variabili con scope di blocco, introdotto in ES6.
let: Simile a let, ma per dichiarare variabili il cui valore non può essere riassegnato.


FUNZIONI

Dichiarazione di una Funzione: Una funzione viene dichiarata con la parola chiave function, seguita dal nome della funzione e da una coppia di parentesi tonde che possono contenere parametr.
Invocazione di una Funzione: Per eseguire una funzione, la si invoca utilizzando il suo nome seguito da parentesi tonde, eventualmente contenenti argomenti.
Valore di Ritorno: Una funzione può restituire un valore al codice che l’ha chiamata utilizzando l’istruzione return.
Funzioni Anonime e Espressioni di Funzione: Le funzioni possono anche essere definite senza un nome, creando così una funzione anonima, spesso utilizzata come valore di callback o assegnata a variabili.
Arrow Functions (Funzioni Freccia): Introdotte in ES6, le arrow functions offrono una sintassi più breve e gestiscono il valore di this in modo diverso.
Funzioni Come Valori di Prima Classe: In JavaScript, le funzioni sono trattate come valori di prima classe, il che significa che possono essere assegnate a variabili, passate come argomenti ad altre funzioni e restituite da altre funzioni.
Funzioni Immediatamente Invocate (IIFE): Un IIFE è una funzione che viene eseguita non appena viene definita

ARRAY

Gli array in JavaScript sono strutture dati molto versatili che ti permettono di memorizzare più valori in un’unica variabile. Ecco alcune caratteristiche e metodi fondamentali degli array in JavaScript:

Creazione di un Array:
Literal:const array = [valore1, valore2, ...];
Constructor: const array = new Array(valore1, valore2, ...);

Accesso agli Elementi:
Accedi a un elemento tramite il suo indice: const elemento = array[indice];
Modifica di un Array:
Mutabili: push(), pop(), splice(), shift(), unshift(), reverse(), sort()
Immutabili: concat(), slice(), map(), filter(), reduce()
Proprietà e Metodi Utili:
length: restituisce la lunghezza dell’array.
join(): unisce tutti gli elementi di un array in una stringa.
indexOf(): restituisce il primo indice al quale un dato elemento può essere trovato nell’array.

Gli array in JavaScript sono oggetti speciali con proprietà e metodi integrati che facilitano la manipolazione dei dati. Sono flessibili e possono contenere una miscela di diversi tipi di dati


ARRAY MUTABILI ED IMMUTABILI

In JavaScript, la differenza tra array mutabili e immutabili riguarda la capacità di modificare i dati dopo che sono stati creati. Gli array mutabili possono essere modificati, mentre quelli immutabili non possono.
Array Mutabili:
Gli array in JavaScript sono di natura mutabile, il che significa che il loro contenuto può essere modificato dopo la creazione. Ecco alcuni metodi mutabili:
push(): aggiunge elementi alla fine dell’array.
pop(): rimuove l’ultimo elemento dall’array.
splice(): può essere utilizzato per aggiungere o rimuovere elementi da un array.
shift(): rimuove il primo elemento dall’array.
unshift(): aggiunge elementi all’inizio dell’array.
reverse(): inverte l’ordine degli elementi nell’array.
sort(): ordina gli elementi dell’array secondo una funzione di confronto.

Array Immutabili:
Per lavorare con array in modo immutabile, si utilizzano metodi che non modificano l’array originale ma restituiscono un nuovo array con le modifiche desiderate

concat(): unisce due o più array.
slice(): estrae una sezione di un array e la restituisce come un nuovo array.
map(): crea un nuovo array con i risultati della chiamata di una funzione fornita su ogni elemento dell’array.
filter(): crea un nuovo array con tutti gli elementi che superano un test implementato da una funzione fornita.
reduce(): applica una funzione contro un accumulatore e ogni valore dell’array (da sinistra a destra) per ridurlo a un singolo valore.

Utilizzare metodi immutabili è una pratica comune nella programmazione funzionale perché aiuta a prevenire effetti collaterali e rende il codice più prevedibile e facile da testare.


OPERATORI

In JavaScript, gli operatori vengono utilizzati per eseguire azioni su variabili e valori. Gli operatori logici, aritmetici e comparativi sono fondamentali per la costruzione di espressioni e la logica di controllo del flusso.

Operatori Logici:

&& (AND): Restituisce true se entrambe le espressioni sono true.
|| (OR): Restituisce true se almeno una delle espressioni è true.
! (NOT): Restituisce true se l’espressione è false, e viceversa.
?? (Nullish Coalescing): Restituisce il primo operando se non è né null né undefined, altrimenti restituisce il secondo.

Operatori Aritmetici:

+ (Addizione): Somma due numeri o concatena stringhe.
- (Sottrazione): Sottrae il secondo numero dal primo.
* (Moltiplicazione): Moltiplica due numeri.
/ (Divisione): Divide il primo numero per il secondo.
% (Modulo): Restituisce il resto di una divisione.
++ (Incremento): Aumenta il valore di una variabile di uno.
-- (Decremento): Diminuisce il valore di una variabile di uno.


Operatori Comparativi:

== (Uguale a): Verifica se due valori sono uguali (conversione di tipo se necessario).
=== (Strettamente uguale a): Verifica se due valori sono uguali e dello stesso tipo.
!= (Non uguale a): Verifica se due valori non sono uguali.
!== (Strettamente non uguale a): Verifica se due valori non sono uguali o non sono dello stesso tipo.
> (Maggiore di): Verifica se il primo valore è maggiore del secondo.
< (Minore di): Verifica se il primo valore è minore del secondo.
>= (Maggiore o uguale a): Verifica se il primo valore è maggiore o uguale al secondo.
<= (Minore o uguale a): Verifica se il primo valore è minore o uguale al secondo.
