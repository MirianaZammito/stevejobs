SELETTORI

Settori di tipo:

si riferiscono a tutti gli elementi di un tipo specifico

      p {
          /* Stili per tutti gli elementi <p> */
      }

Selettori di classe:

Si riferiscono agli elementi che hanno specificate una o piÃ¹ classi. Si utilizzano il punto "." seguito dal nome della classe. 

      .classe {
          /* Stili per gli elementi con la classe "classe" */
      }

Selettori di ID:

Si riferiscono a un elemento specifico con un ID univoco. Si utilizza il carattere "#" seguito dal nome dell'ID. 

      #id {
          /* Stili per l'elemento con l'ID "id" */
      }

Selettori discendenti:

Selezionano elementi che sono discendenti diretti di un altro elemento. Si usano gli spazi per separare gli elementi. 

      <div> p {
      /* Stili per tutti gli elementi <p> che sono discendenti diretti di un elemento<div>  */
      }

Selettori figlio:

Selezionano gli elementi che sono figli diretti di un altro elemento. Si usa il simbolo ">" per separare il genitore dal figlio. 

      <div> p {
      /* Stili per tutti gli elementi <p> che sono figli diretti di un elemento <div>  */
      }

Selettori attributo:

Selezionano gli elementi che hanno un determinato attributo specificato. Si utilizzano le parentesi quadre e l'operatore "=". Ad esempio, [type="text"] selezionerÃ  tutti gli elementi con l'attributo type uguale a "text". 

      [type="text"] {
          /* Stili per gli elementi con l'attributo type="text" */
      }

Selettori universali:

Selezionano tutti gli elementi sul documento. Si usa il carattere "\*". 

      * {
          /* Stili per tutti gli elementi */
      }
